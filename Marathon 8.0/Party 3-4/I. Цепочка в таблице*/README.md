# I. Цепочка в таблице*

## Описание задачи

Дана прямоугольная таблица размером `n × m`, заполненная целыми числами.  
Необходимо найти **максимальную длину цепочки** из **последовательных натуральных чисел**, расположенных в **соседних по стороне** клетках таблицы.

> Соседними считаются только клетки, имеющие общую **сторону** — движение разрешено **вверх**, **вниз**, **влево** и **вправо**.  
> Диагональное перемещение **запрещено**.

---

## Формат ввода

В первой строке записаны два числа:
— размеры таблицы, где  
`1 ≤ n, m ≤ 1000`.

В следующих `n` строках содержится по `m` целых чисел `a[i][j]` — элементы таблицы:
1 ≤ a[i][j] ≤ 10 × n × m

---

## Формат вывода

Выведите одно число — максимальную длину цепочки последовательных чисел в таблице.
