# D. Словарь

## Описание задачи

У Васи на клавиатуре не работает клавиша пробел, поэтому все тексты он теперь набирает слитно.  
Необходимо написать программу, которая разделяет текст Васи на слова из данного словаря.

Программа должна корректно разбивать текст на слова так, чтобы получился читаемый вариант.  
Если возможны несколько вариантов разбиения строки на слова, разрешается выводить любой из них.  
Гарантируется, что хотя бы один способ разбиения существует.

---

## Формат ввода

1. Первая строка — текст, введённый Васей, **без пробелов**.  
   - Содержит не более 100 латинских строчных букв (`a-z`).

2. Вторая строка — число `N` (`1 ≤ N ≤ 2000`) — количество слов в словаре.

3. Следующие `N` строк — слова словаря, по одному слову в строке.  
   - Каждое слово содержит не более 20 латинских строчных букв.  
   - Слова **записаны в алфавитном порядке**.

---

## Формат вывода

- Выведите текст Васи с пробелами между словами.  
- Пробел после последнего слова **допустим**, но необязателен.
