# B. Поход

## Условие

Группа школьников планирует поход вдоль Москвы-реки с левого берега на правый. По пути встречаются притоки, впадающие с левого (L), правого (R) берега или одновременно с обоих (B). 

**Цель:** определить минимальное количество переправ через реку и притоки, необходимых для достижения правого берега.

## Формат ввода
Одна строка длиной до 200 символов, содержащая последовательность букв:
- `L` - приток впадает слева
- `R` - приток впадает справа  
- `B` - притоки впадают с обоих берегов

## Формат вывода
Одно целое число - минимальное количество переправ.

## Пример
Ввод: LRLR
Вывод: 2
