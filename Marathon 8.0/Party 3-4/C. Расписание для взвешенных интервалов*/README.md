# C. Расписание для взвешенных интервалов*

Дан набор временных интервалов, у каждого из которых задан вес. Множество интервалов называется совместимым, если никакие два интервала в нем не пересекаются. Под временными интервалами подразумеваются интервалы вида [b, e), не включающие свои концы. Необходимо найти совместимое подмножество интервалов с максимальным суммарным весом.

Формат ввода:
В первой строке записано количество интервалов N (0 ≤ N ≤ 10^5). В следующих N строках для каждого интервала указаны три вещественных числа: начало интервала b_i, конец интервала e_i (b_i < e_i) и вес w_i (w_i > 0).

Формат вывода:
Выведите одно число — максимальную суммарную сумму весов совместимого подмножества интервалов. Ответ считается правильным, если отличается от точного не более чем на 10^-4.
