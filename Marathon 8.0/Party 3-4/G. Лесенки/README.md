# G. Лесенки

## Задача

Лесенкой называется набор кубиков, в котором каждый горизонтальный слой содержит **строго меньше кубиков**, чем слой под ним. В нижнем слое может быть любое количество кубиков.

Необходимо подсчитать количество различных лесенок, которые могут быть построены из **N** кубиков.

## Примеры

### N = 5
Возможные лесенки:
- [5] (один слой)
- [4, 1] (4 внизу, 1 сверху)
- [3, 2] (3 внизу, 2 сверху)

**Ответ: 3**

### N = 7
Возможные лесенки:
- [7]
- [6, 1]
- [5, 2]
- [4, 3]
- [4, 2, 1]

**Ответ: 5**

## Математическая формулировка

Задача эквивалентна нахождению количества **разбиений числа N на различные натуральные слагаемые**.

Согласно **теореме Эйлера**, количество разбиений на различные слагаемые равно количеству разбиений на нечётные слагаемые.

Поэтому находим кол-во разбиений числа на нечетные слагаемые - это и будет кол-во лестниц.
