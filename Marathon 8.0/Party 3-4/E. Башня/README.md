# E. Башня

## Условие

Петя построил крепость из кубиков, которая представляет собой последовательность из **N** столбиков. Высота **i**-го столбика равна **Aᵢ**.

Для оценки защищённости крепости Петя ввёл понятие **башни**. Башней называется любая последовательность из **K** идущих подряд столбиков (где **K** — любимое число Пети).

**Защищённость** одной башни вычисляется по формуле:

> `(сумма высот всех столбиков башни) × (минимальная высота столбика в башне)`

**Неприступность крепости** определяется как сумма защищённостей всех башен. При этом столбики должны быть разбиты на непересекающиеся башни (каждый столбик принадлежит ровно одной башне).

Петя выбирает такое разбиение на башни, при котором неприступность крепости **максимальна**.
