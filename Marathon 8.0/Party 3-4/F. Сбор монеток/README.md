# F. Сбор монеток

## Условие задачи

В компьютерной игре игрок двигается по дороге сверху-вниз. Цель игры — собрать как можно больше монеток. 

Дорога представляет собой таблицу размером `n × 3` клеток. Каждая клетка может быть:
- **"."** — пустая
- **"C"** — содержит монетку
- **"W"** — стена

Игрок начинает движение с первой строки и может выбрать любую из трех начальных клеток. Движение продолжается до тех пор, пока игрок не врежется в стену или не достигнет конца дороги (последней строки `n`).

## Правила движения

Если игрок находится в клетке `(r, c)`, то за один ход он может переместиться в одну из трех клеток:
- `(r+1, c-1)` — вниз-влево
- `(r+1, c)` — прямо вниз
- `(r+1, c+1)` — вниз-вправо

Перемещение возможно только если:
- Клетка находится в пределах дороги (столбцы 0-2)
- Клетка не содержит стены

## Особые условия

Если в первой строке во всех клетках находятся стены, игра заканчивается сразу.

## Формат ввода

Первая строка: число `n` (`1 ≤ n ≤ 10⁴`) — количество строк в таблице

Следующие `n` строк: по три символа, описывающих строку таблицы

## Формат вывода

Одно число — максимальное количество монеток, которое можно собрать.
