# B. Из тупика в тупик

## Условие задачи

Каждый вечер после работы Вася приходит домой и думает о жизни. Вася описал свои перспективы в виде `n` жизненных ситуаций и соединил их `n-1` двунаправленными переходами так, что между двумя любыми жизненными ситуациями существовал ровно один простой путь, возможно через промежуточные ситуации.

**Жизненным тупиком** называется жизненная ситуация, из которой существует ровно один переход.

Помогите Васе найти самый короткий путь из одного жизненного тупика в другой.

## Формат входных данных

- Первая строка: целое число `n` (2 ≤ n ≤ 10^5) - количество жизненных ситуаций
- Следующие `n-1` строк: пары целых чисел `a b` (1 ≤ a, b ≤ n) - переходы между ситуациями

## Формат выходных данных

- Одно целое число - длина кратчайшего пути между двумя жизненными тупиками
