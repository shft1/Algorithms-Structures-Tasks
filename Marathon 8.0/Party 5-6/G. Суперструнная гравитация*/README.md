# G. Суперструнная гравитация*

## Условие задачи

Недавно учёные открыли **суперструнную гравитацию** — силу, вычисляемую необычным образом.  

В пространстве существуют две частицы — **A** и **B**.  
Каждая из них живёт в своём многомерном пространстве и имеет собственные координаты.

- Частица **A** существует в *n*-мерном пространстве и имеет координаты:  
  `a₁, a₂, …, aₙ`

- Частица **B** существует в *m*-мерном пространстве и имеет координаты:  
  `b₁, b₂, …, bₘ`

Сила гравитации между двумя частицами вычисляется по формуле:

\[
F = \sum_{i=1}^{n} \sum_{j=1}^{m} (i - j) \cdot |a_i - b_j|
\]

---

## Формат входных данных

- Первая строка содержит число  
  `n` — количество измерений пространства частицы **A**  
  *(1 ≤ n ≤ 10⁵)*  

- Вторая строка содержит `n` чисел `a₁, a₂, …, aₙ` — координаты частицы **A**  
  *(1 ≤ aᵢ ≤ 10⁴)*  

- Третья строка содержит число  
  `m` — количество измерений пространства частицы **B**  
  *(1 ≤ m ≤ 10⁵)*  

- Четвёртая строка содержит `m` чисел `b₁, b₂, …, bₘ` — координаты частицы **B**  
  *(1 ≤ bⱼ ≤ 10⁴)*  

---

## Формат выходных данных

Выведите одно число — силу гравитации между частицами **A** и **B**.
