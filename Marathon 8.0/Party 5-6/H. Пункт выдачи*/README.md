# H. Пункт выдачи*

## Условие

Город Деревянск состоит из `n` площадей, соединенных двусторонними дорогами таким образом, что от любой площади до любой другой можно доехать единственным способом (граф представляет собой дерево).

На каждой площади `i` проживает `a_i` человек.

Планируется открыть пункт выдачи заказов на одной из площадей. При этом:

- Для жителей площади, на которой откроется пункт, будет отдельный вход
- Со стороны каждой дороги, ведущей на другие площади, также будут построены отдельные входы
- К каждому входу выстроится очередь из всех жителей, живущих "со стороны этой двери"

**Цель**: выбрать площадь для открытия пункта выдачи так, чтобы длина максимальной очереди была минимальна.

## Входные данные

- Первая строка: `n` (1 ≤ n ≤ 100,000) - количество площадей
- Вторая строка: `n` чисел `a_i` (1 ≤ a_i ≤ 10^9) - количество жителей на каждой площади
- Следующие `n-1` строк: пары чисел `v_j, u_j` (1 ≤ v_j, u_j ≤ n) - номера соединенных дорогой площадей

## Выходные данные

- Одно число - номер площади, на которой нужно построить пункт выдачи (если ответов несколько, вывести любой)
