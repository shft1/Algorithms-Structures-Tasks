# F. Предок

## Описание задачи

Дано корневое дерево из `n` вершин, где вершина `0` является корнем. Для каждой вершины `i` от `1` до `n-1` известен её непосредственный родитель.

Необходимо обработать `m` запросов вида `(a, b)`, где каждый запрос проверяет, является ли вершина `a` предком вершины `b` в дереве.

## Формат входных данных

- Первая строка: целое число `n` — количество вершин в дереве (`1 ≤ n ≤ 100,000`)
- Вторая строка: `n-1` целых чисел через пробел — родители вершин `1` до `n-1`
- Третья строка: целое число `m` — количество запросов (`1 ≤ m ≤ 100,000`)
- Следующие `m` строк: по два целых числа `a` и `b` через пробел — параметры запросов

## Формат выходных данных

- `m` строк, по одной на запрос: 
  - `1`, если `a` является предком `b`
  - `0` в противном случае
