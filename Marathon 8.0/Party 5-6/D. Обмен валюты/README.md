# D. Обмен валюты

## Условие задачи

Феоктист работает в обменном пункте на границе Флатландии и Байтландии. Каждый день он узнает по радио текущий курс обмена флатов на биты и должен вывесить две таблички с числами так, чтобы их отношение было максимально близко к курсу.

В распоряжении Феоктиста есть `n` табличек с числами `c₁, c₂, ..., cₙ`. Узнав сегодняшний курс обмена `p`, он выбирает две таблички с значениями `cᵢ` и `cⱼ` такие, что величина `|cᵢ/cⱼ - p|` минимальна, и вывешивает их в виде объявления «меняю cᵢ флатов на cⱼ битов».

## Формат ввода

- `n` (2 ≤ n ≤ 100,000) - количество табличек
- `p` (1 ≤ p ≤ 10⁹) - текущий курс обмена
- `cᵢ` (1 ≤ cᵢ ≤ 10⁹) - числа на табличках

## Формат вывода


Два целых числа - номера табличек (индексы от 1 до n), таких что `|cᵢ/cⱼ - p|` минимально. Если таких пар несколько, можно вывести любую.
