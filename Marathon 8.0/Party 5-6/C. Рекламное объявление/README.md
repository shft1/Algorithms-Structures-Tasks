# C. Рекламное объявление

## Условие задачи

Ивану нужно сверстать рекламное объявление, состоящее из N слов в заданном порядке. Объявление должно размещаться на поле шириной W и высотой H.

## Требования к верстке

1. **Одинаковый масштаб**: Все слова должны быть напечатаны в одном масштабе `k`
2. **Размеры слов**: Изначально i-е слово имеет размеры `a_i × b_i`. В масштабе `k` оно занимает `(k·a_i) × (k·b_i)`
3. **Высота строк**: Если в строке более одного слова, все слова в ней должны иметь одинаковую высоту
4. **Порядок слов**: Слова должны располагаться в заданном порядке (сверху вниз, слева направо)
5. **Границы поля**: Ни одно слово не должно выходить за границы поля

## Цель

Найти максимальный масштаб `k`, при котором можно сверстать объявление, удовлетворяющее всем требованиям.
