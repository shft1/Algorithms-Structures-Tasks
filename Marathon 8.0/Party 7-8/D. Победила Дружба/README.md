# D. Победила Дружба

## Условие задачи

Вася и Маша участвуют в соревновании по поеданию сырков «Дружба». На `n` столах, расставленных в ряд и занумерованных с `1` по `n`, лежат сырки. На `i`-м столе расположено `aᵢ` сырков.

### Правила соревнования:
- Если участник подходит к столу, то он должен съесть **все** сырки с этого стола
- **Вася** начинает слева и ест сырки на столах с `1` по `l` включительно
- **Маша** начинает справа и ест сырки на столах с `n` по `r` включительно
- При этом гарантируется, что `l < r` (столы не пересекаются)

### Цель:
Победить должна **дружба** — количество съеденных сырков у Васи (`SDᵥ`) и Маши (`SDₘ`) должно отличаться как можно меньше.
