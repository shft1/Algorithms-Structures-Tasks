# H. Премии от начальника*

## Краткое описание

Начальник намерен выдать премии сотрудникам. Все сотрудники пронумерованы в порядке **возрастания профессионализма** от 0 до n−1. Для каждого сотрудника i задано целое число `a_i` — количество выходных дней, которые он отработал.

Для каждого сотрудника `i` премия вычисляется так:

1. Считается количество сотрудников `j` с `j < i`, для которых выполняется условие `j + a_j > i`.
2. Полученное количество умножается на `a_i` — это и есть премия сотрудника `i`.

Требуется найти суммарный размер премий для всех сотрудников.

> Замечание: индексация сотрудников в формулировке задачи может быть представлена как 0-based (от 0 до n−1). Если в конкретной реализации/вводе используется 1-based, нужно аккуратно согласовать индексы.

---

## Формат ввода

Первая строка содержит одно целое число `n` (1 ≤ n ≤ 100000) — количество сотрудников.

Вторая строка содержит `n` целых чисел `a_0, a_1, ..., a_{n-1}` (0 ≤ a_i ≤ n) — количество выходных дней, которые отработал i-й сотрудник.

Числа разделяются пробелами.

---

## Формат вывода

Выведите одно целое число — суммарный размер премий всех сотрудников.

---

## Пример

**Ввод**

```
6
1 2 0 1 1 0
```

**Вывод**

```
5
```
