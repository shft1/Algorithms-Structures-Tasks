# B. Транспортные налоги

## Условие

Необходимо рассчитать транспортный налог для автомобилей на основе мощности их двигателя в лошадиных силах и таблицы налоговых ставок.

### Ограничения

- 1 ≤ n ≤ 100,000
- 1 ≤ m ≤ 100,000  
- 0 ≤ bᵢ, tᵢ ≤ 10⁹
- 1 ≤ qⱼ ≤ 10⁹
- b₁ = 0
- bᵢ < bᵢ₊₁ (мощности в таблице строго возрастают)
- tᵢ ≤ tᵢ₊₁ (ставки не убывают)

### Особенности расчета:

- Налог рассчитывается как: `налог = ставка × мощность`
- Таблица ставок задает диапазоны мощностей
- Ставка из i-й строки применяется для мощностей, которые:
  - Строго больше мощности в текущей строке
  - Меньше или равны мощности в следующей строке (если она существует)
- Если мощность превышает максимальную в таблице, применяется максимальная ставка
