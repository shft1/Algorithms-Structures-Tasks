# E. Ремонт выбоин

## Условие

На прямой трассе расположено `n` участков, пронумерованных от `1` до `n` слева направо. По трассе проложено `m` автобусных маршрутов. Каждый маршрут `i` проходит по участкам с `l_i` по `r_i` включительно.

На каждом участке `j` изначально находится `a_j` выбоин. **Дискомфорт** от поездки по маршруту равен суммарному количеству выбоин на всех участках, по которым проходит этот маршрут.

Автодорожное предприятие имеет бюджет на ремонт не более `k` выбоин. При этом:
- Количество выбоин на участке не может стать отрицательным
- Можно ремонтировать выбоины на любых участках в любом количестве (в пределах бюджета)

**Цель:** определить минимальный суммарный дискомфорт от поездок по всем маршрутам после оптимального распределения ремонтных работ.
