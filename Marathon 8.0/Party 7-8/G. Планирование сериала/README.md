# G. Планирование сериала

## Условие задачи

Вася снимается в популярном сериале и вместе со сценаристами составил план с количеством серий на ближайшие `n` сезонов (один сезон снимают за один месяц).

Для каждого сезона `i` известно:
- `sᵢ` — запланированное количество серий;
- `aᵢ` — коэффициент важности других дел Васи на время съёмки этого сезона.

Однако концепция сериала изменилась — теперь в каждом сезоне должно быть одинаковое количество серий.  
Если в каждом сезоне будет по `e` серий, то для сезона `i` разница с планом составит:
dᵢ = |e - sᵢ|
За каждое такое изменение Вася требует компенсацию в размере:
dᵢ × aᵢ
Общая доплата за все сезоны равна:
Σ (aᵢ × |e - sᵢ|)
---

## Задача

Найдите такое количество серий `e`, чтобы суммарная доплата Васе за все сезоны была минимальной.  
Если существует несколько вариантов выбора `e`, дающих одинаковую минимальную сумму, выберите наименьшее `e`.

---

## Формат ввода

- В первой строке вводится число `n` — количество запланированных сезонов.  
- Во второй строке указаны `n` чисел `s₁, s₂, ..., sₙ` — запланированное количество серий.  
- В третьей строке указаны `n` чисел `a₁, a₂, ..., aₙ` — коэффициенты важности других дел Васи.

---

## Формат вывода

Выведите два числа:
1. Количество серий `e` в каждом сезоне;
2. Минимальную сумму доплаты Васе.

Если существует несколько оптимальных значений `e`, выведите наименьшее из них.
