# A. Рейсы между офисами

## Условие

У компании есть два офиса, между которыми необходимо организовать автобусное сообщение. Расписание уже составлено, но требуется определить минимальное количество автобусов, достаточное для обслуживания всех рейсов.

### Особенности задачи:

- Расписание состоит из **N** рейсов из первого офиса во второй и **M** обратных рейсов
- Все рейсы выполняются в течение одних календарных суток (с 00:00 до 23:59)
- Ни один рейс не прибывает на следующий день после отправления
- Минимальная продолжительность рейса - 1 минута (рейс не может прибыть в ту же минуту, когда отправился)
- После прибытия автобус готов к отправлению в обратный рейс в ту же минуту
- Количество автобусов у офиса не ограничено
- Автобусы не могут перемещаться вне расписания
