# J. Листы и вью*

Описание задачи

Необходимо реализовать интерпретатор языка «Фигон», который поддерживает работу со списками чисел и позволяет присваивать отдельные имена как самим спискам, так и их частям. Язык поддерживает пять основных операций: создание нового списка, создание подсписка, изменение элемента списка, добавление элемента в список и получение элемента списка.

Операция создания нового списка имеет вид List a = new List(x,y,...,z) и создаёт новый список чисел, присваивая ему имя a.

Операция создания подсписка имеет вид List b = a.subList(from,to) и создаёт новый список b, состоящий из элементов списка a, находящихся на позициях от from до to включительно. При этом подсписок не является независимой копией: все изменения, вносимые в элементы исходного списка a, отражаются в подсписке b, и наоборот. Однако подсписки нельзя расширять — операция добавления новых элементов для них не поддерживается.

Операция изменения элемента списка записывается как a.set(i,x) и присваивает i-му элементу списка a новое значение x.

Операция добавления элемента в конец списка записывается как a.add(x) и добавляет число x в конец списка a. Эта операция допустима только для списков, созданных напрямую через new List(...), и не может применяться к подспискам.

Операция получения элемента имеет вид a.get(i) и выводит значение i-го элемента списка a в отдельной строке.

Во входных данных сначала задаётся число n — количество строк программы. Затем следуют n строк, каждая из которых представляет одну из описанных операций. Гарантируется, что никакой список не объявляется дважды, обращения по индексам не выходят за границы списка, а суммарная длина всех списков, созданных через new List(...), не превышает 100000. Все числа в программе — целые положительные значения, не превосходящие 10⁹.

Для каждой операции a.get(i) необходимо вывести значение соответствующего элемента списка.
