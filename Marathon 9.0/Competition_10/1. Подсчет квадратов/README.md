# 1. Подсчет квадратов

## Решение

Для решения задачи нужно найти все квадраты из символов `#` со сторонами, параллельными краям снимка.  
Квадраты могут быть разных размеров (1×1, 2×2, …), но не должны пересекаться или соприкасаться — гарантируется, что бактерии не соприкасаются.  
Таким образом, каждое связное множество `#` образует ровно один квадрат.

Алгоритм:
1. Проверяем, что `n` и `m` не отрицательны — иначе выводим `-2`.
2. Проходим по всем клеткам.
3. Если находим `#`, определяем размер квадрата, начиная с минимальной стороны.
4. Увеличиваем размер, пока все клетки в этом квадрате — `#`, и квадрат не выходит за границы.
5. После нахождения максимального квадрата удаляем его (заменяем на `.`) и увеличиваем счётчик.
6. Повторяем, пока есть `#`.
