# 2. Телеметрия

## Описание задачи

Есть `n` окон, в каждом из которых отображаются последние `k` символов текстового документа.  
Изначально все документы пусты, активное окно — первое.  
Буфер обмена изначально пуст.

Последовательность действий:

- **Строчная латинская буква** — добавить символ в конец текущего документа.
- **Backspace** — удалить последний символ из текущего документа (если документ не пуст).
- **Copy** — сохранить в буфер обмена текст, отображаемый в текущем окне (последние `k` символов документа).
- **Paste** — добавить содержимое буфера обмена в конец текущего документа.
- **Next** — переключиться на следующее окно (после последнего — первое).

После выполнения всех команд нужно вывести содержимое **текущего окна** (последние `k` символов) или `Empty`, если оно пусто.

## Ограничения

- `1 ≤ n ≤ 10`
- `1 ≤ m, k ≤ 1000`
- Если `n < 0` или `m < 0` или `k < 0`, вывести `-2`

## Алгоритм

1. Проверить входные параметры на отрицательность.
2. Создать список из `n` строк (документы), изначально пустые.
3. Переменная `current` = 0 (индекс активного окна).
4. Буфер обмена `clipboard` = пустая строка.
5. Для каждой команды:
   - **Буква** → добавить в документ[current].
   - **Backspace** → удалить последний символ, если строка не пуста.
   - **Copy** → `clipboard` = последние `k` символов документа[current].
   - **Paste** → добавить `clipboard` в документ[current].
   - **Next** → `current = (current + 1) % n`.
6. После всех команд взять документ[current], обрезать до последних `k` символов.
7. Если результат пуст → вывести `Empty`, иначе вывести его.
