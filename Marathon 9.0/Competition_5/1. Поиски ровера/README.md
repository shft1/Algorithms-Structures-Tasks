## Поиск ровера

Нам нужно найти максимальное значение \( X \) (координата ровера), которое удовлетворяет всем условиям:

\[
|X - x_i| \leq d_i \quad \text{для всех } i = 1 \dots n
\]

Это эквивалентно системе неравенств:

\[
x_i - d_i \leq X \leq x_i + d_i \quad \forall i
\]

Таким образом, \( X \) должно лежать в пересечении всех отрезков \([L_i, R_i]\), где:

\[
L_i = x_i - d_i, \quad R_i = x_i + d_i
\]

### Шаги решения

1. Для каждой базовой станции вычислим:
   - \( L_i = x_i - d_i \)
   - \( R_i = x_i + d_i \)

2. Найдем пересечение всех отрезков:
   - \( L_{\max} = \max(L_1, L_2, \dots, L_n) \)
   - \( R_{\min} = \min(R_1, R_2, \dots, R_n) \)

3. Если \( L_{\max} > R_{\min} \), пересечения нет → ответ `-1`.

4. Если \( L_{\max} \leq R_{\min} \), максимальное возможное \( X \) равно \( R_{\min} \).
