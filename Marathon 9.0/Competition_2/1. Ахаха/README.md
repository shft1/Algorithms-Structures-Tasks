# Задача: Поиск самой длинной подстроки-смеха

## Условие

Смех — это строка, состоящая только из чередующихся символов `'a'` и `'h'`.  
Примеры смеха: `"ahahaha"`, `"hah"`, `"a"`, `"h"`.  
Неподходящие строки: `"abacaba"`, `"hh"`, `"aa"`.

Дана строка длины `n` (до 10⁵). Нужно найти **длину самой длинной подстроки**, которая является смехом.
