# Разбор задачи: Упаковка простыни

## Условие

Даны размеры простыни `n × m` и размеры пакета `h × w`. Простыню можно:
- Складывать пополам (по горизонтали или вертикали) любое количество раз
- Поворачивать на 90 градусов

Стороны сложенной простыни должны быть параллельны сторонам пакета, а её размеры не должны превышать размеры пакета в соответствующих измерениях.

Нужно найти **минимальное количество складываний**.
