# Расстановка кораблей

Согласно правилам игры «Морской бой», у игрока есть поле размера `10 x 10`, на котором нужно расставить `10` кораблей:

- один четырехпалубный корабль размера `1 x 4`;
- два трехпалубных корабля размера `1 x 3`;
- три двухпалубных корабля размера `1 x 2`;
- четыре однопалубных корабля размера `1 x 1`.

При этом должны выполняться следующие условия:

- на поле должны быть выставлены все корабли;
- каждый корабль целиком помещается внутри поля;
- клетки каждого корабля образуют прямоугольник соответствующего размера;
- каждый корабль расположен либо вертикально, либо горизонтально;
- любые две клетки разных кораблей не могут совпадать и не могут касаться друг друга ни по стороне, ни по углу.

Дана таблица `10 x 10`, где символ `"#"` обозначает занятую клетку, а `"."` — пустую.

Необходимо определить, является ли расстановка кораблей корректной.

## Формат ввода

Ввод состоит из `10` строк по `10` символов в каждой. Каждая строка описывает одну строку поля.

Гарантируется, что каждый символ равен либо `"#"`, либо `"."`.

## Формат вывода

Выведите `YES`, если поле соответствует корректной расстановке кораблей, и `NO` иначе.

## Пример 1

### Ввод

```text
#.#.#.....
.......##.
.#..#.....
.#........
.#..##....
..........
####...#..
.......#..
.......#..
##........
```

### Вывод

```text
YES
```

## Пример 2

### Ввод

```text
##..#.....
.......##.
.#..#....#
.#........
.#........
..........
####...#..
.......#..
.......#..
##.......#
```

### Вывод

```text
NO
```
