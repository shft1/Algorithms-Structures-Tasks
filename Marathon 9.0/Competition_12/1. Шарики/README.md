# Шарики

В компьютерной игре игрок выставляет в линию шарики разных цветов. Когда образуется непрерывная цепочка из трех и более шариков одного цвета, она удаляется из линии. После этого все оставшиеся шарики сдвигаются друг к другу, и ситуация может повториться.

Необходимо определить, сколько шариков будет уничтожено.

Гарантируется, что в начальный момент непрерывных цепочек из трех и более одноцветных шаров может быть не более одной.

## Формат ввода

В одной строке записано количество шариков в цепочке и их цвета.

- количество шариков не превышает `1000`;
- цвет каждого шарика задается целым числом от `0` до `9`.

## Формат вывода

Выведите количество шариков, которое будет уничтожено.

## Пример 1

### Ввод

```text
5 1 3 3 3 2
```

### Вывод

```text
3
```

## Пример 2

### Ввод

```text
10 3 3 2 1 1 1 2 2 3 3
```

### Вывод

```text
10
```

## Пример 3

### Ввод

```text
3 0 0 0
```

### Вывод

```text
3
```
