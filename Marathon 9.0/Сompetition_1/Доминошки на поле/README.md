# Задача 1. Доминошки на поле

## Условие

Дано прямоугольное клетчатое поле размером `n × m`.

Некоторые клетки заняты, остальные — свободны.  
Требуется определить, **сколькими способами можно разместить одну доминошку размером `2 × 1`** на этом поле.

Доминошка:
- занимает ровно две соседние клетки;
- может быть размещена:
  - горизонтально (влево-вправо),
  - вертикально (вверх-вниз).

### Ограничения
Доминошку можно разместить только если:
- обе клетки находятся внутри поля;
- обе клетки **свободны** (не заняты).
