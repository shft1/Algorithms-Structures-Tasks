# 1. Закраска клеток

## Условие

Робот находится на **бесконечном клетчатом поле**.  
Программа для робота задаётся строкой из команд:

- `U` — сдвинуться на **одну клетку вверх**
- `R` — сдвинуться на **одну клетку вправо**
- `D` — сдвинуться на **одну клетку вниз**
- `L` — сдвинуться на **одну клетку влево**

Робот **закрашивает все клетки**, на которых он побывал, включая **начальную** клетку.

Требуется определить, **сколько клеток будут закрашены более одного раза** (т.е. посещены дважды или более).

## Формат ввода

В единственной строке входных данных задаётся **строка с программой** для робота.

- Строка состоит только из символов: `U`, `R`, `D`, `L`
- Длина строки не превосходит **1000**

## Формат вывода

Выведите одно целое число — **количество клеток**, которые робот посетил **более одного раза**.
