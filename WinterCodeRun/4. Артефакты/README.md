# 4. Артефакты

Кодерун спускается в Синтаксическую шахту, чтобы изучить тайны старого кода перед восхождением. Иногда в шахте попадаются артефакты — натуральные числа, которые имеют больше натуральных делителей, чем каждое из меньших натуральных чисел.

Нужно обработать q запросов, каждый из которых характеризуется двумя целыми числами l, r. Для каждого запроса найдите количество артефактов на отрезке [l; r].

## Формат ввода

В первой строке содержится единственное целое число q (1 ≤ q ≤ 5·10⁴) — количество запросов.

В последующих q строках содержатся по два числа l, r (1 ≤ l ≤ r ≤ 10¹⁸) — границы отрезков.

## Формат вывода

Для каждого запроса в отдельной строке выведите одно целое число — количество артефактов на отрезке [l; r].

## Ограничения

- Ограничение времени: 1 с
- Ограничение памяти: 64 МБ

## Примеры

### Пример 1

**Ввод:**
```
3
1 1
5 10
6 8
```

**Вывод:**
```
1
1
1
```

## Примечание

В первом запросе 1 является артефактом, поскольку имеет единственный натуральный делитель, и натуральных чисел меньше нет.

Во втором и третьем запросе единственным артефактом является число 6 с 4 делителями: 1, 2, 3, 6. Можно показать, что других артефактов на отрезке нет. 8 не является артефактом, поскольку оно имеет только 4 натуральных делителя — столько же, сколько у числа 6.
