# 3. Снаряжение

## Условие задачи

В тренировочном лагере Кодерун распределяет экипировку между командами. Всего есть **M** единиц снаряжения и **N** групп альпинистов, каждая из которых требует определённое количество единиц экипировки для своих тренировок.

Однако запас экипировки ограничен — снаряжения не хватает, чтобы полностью укомплектовать все группы. Для каждой группы недостача приводит к проблемам в тренировках. **Уровень проблем для группы равен квадрату недостающего количества единиц экипировки**.

Нужно распределить снаряжение так, чтобы **минимизировать суммарное количество проблем**.

---

## Формат входных данных

В первой строке вводятся числа:
- **M** (`1 ≤ M ≤ 2×10⁹`) — количество единиц снаряжения
- **N** (`1 ≤ N ≤ 10⁵`) — количество групп

Во второй строке вводится **N** чисел:
- **Wᵢ** (`1 ≤ Wᵢ ≤ 2×10⁹`) — потребность каждой группы в снаряжении

**Гарантируется**, что суммарная потребность строго больше доступного количества:  
`Σ Wᵢ > M`

---

## Формат выходных данных

Выведите **минимальное суммарное количество "проблем подготовки"** по модулю **10⁹ + 7**.

---

## Пример 1

### Ввод
5 3
3 3 3

text

### Объяснение
- Всего экипировки: **5** единиц
- Групп: **3**, каждой нужно по **3** единицы
- Суммарная потребность: `3 + 3 + 3 = 9 > 5` (недостача)
- Оптимальное распределение: `[2, 2, 1]`
- Проблемы: `(3-2)² + (3-2)² + (3-1)² = 1 + 1 + 4 = 6`
- Ответ: **6 mod (10⁹+7) = 6**

### Вывод
6
