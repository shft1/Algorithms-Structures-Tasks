# G. Веселые пути

**Тип задачи:** DFS на дереве, хранение последних вхождений значений

**Условие:**  
Дано дерево (n ≤ 2·10⁵), на каждой вершине число. Путь от 1 до v называется весёлым, если на нём есть хотя бы две вершины с одинаковыми числами. Для каждой вершины v вывести YES/NO.

**Решение:**  
Запускаем DFS от 1.  
Храним для каждого значения aᵢ последнюю вершину (глубину), где оно встречалось на текущем пути.  
Если при переходе в v число aᵥ уже встречалось на пути от 1 до текущей вершины — путь до v весёлый.  
Это свойство наследуется: если путь до u весёлый, то и до всех его потомков тоже.  
Можно передавать флаг.

**Сложность:** O(n)
