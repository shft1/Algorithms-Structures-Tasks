# C. Плохая строка

**Тип задачи:** Построение перестановки с запрещёнными соседними парами

**Условие:**  
Дана строка s из различных строчных латинских букв (длина ≤ 26). Нужно переставить буквы так, чтобы ни одна упорядоченная пара соседних букв не встречалась в исходной строке как соседняя. Вывести любую такую перестановку.

**Решение:**  
Построим граф, где вершины — буквы, рёбра — запрещённые пары (s[i], s[i+1]).  
Нужно найти гамильтонов путь в графе без запрещённых рёбер.  
Так как букв мало, можно перебрать все перестановки (26! — слишком много, но на практике можно искать жадным построением).  
Более простое решение: заметим, что если букв ≤ 26, то можно упорядочить их в порядке, не содержащем исходных пар, например, разбить на две группы и чередовать.  
Часто работает жадный DFS с возвратом.

**Сложность:** O(26!) в худшем, но на малых данных работает быстро.

Ввод
mirea
Вывод
marie
