# D. Теория куч

**Тип задачи:** Моделирование, цикличность

**Условие:**  
Даны две кучи: n и m, число x (1 ≤ x ≤ max(n, m)) и количество операций q (до 10¹²).  
За операцию: выбрать кучу с большим количеством (при равенстве — первую), переложить из неё x в другую. Вывести размеры куч после q операций.

**Решение:**  
Прямое моделирование невозможно при q до 10¹².  
Заметим, что состояние (n, m) циклично, так как сумма n+m постоянна, а разность меняется на 2x или 0.  
Разность d = n−m.  
Если d ≥ x, то n уменьшается на x, m увеличивается на x → d уменьшается на 2x.  
Если d < 0 и |d| ≥ x, то m уменьшается на x, n увеличивается на x → d увеличивается на 2x.  
Если |d| < x, то кучи меняются местами (или нет при равенстве).  
Так как состояния конечны (n+m ≤ 2·10⁶), можно моделировать до первого повторения, а затем использовать остаток от q.

**Сложность:** O(n+m) на поиск цикла.
