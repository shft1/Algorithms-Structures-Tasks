# F. Многомяч

**Тип задачи:** Жадный алгоритм, динамическое программирование

**Условие:**  
На полоске из n клеток (1..n) в каждой клетке один игрок и один мяч. Игрок в клетке i может бросить мяч на 1..aᵢ клеток влево (если бросок выходит за клетку 1 — мяч вылетает). Задача: выкинуть все мячи за клетку 1 за минимальное количество бросков.

**Решение:**  
Справа налево:  
Если в клетке i есть мяч, нужно его бросить. Лучше всего бросить на максимальную дальность, чтобы заодно выбить мячи из промежуточных клеток.  
Используем жадность: поддерживаем массив, сколько бросков нужно сделать, начиная с конца.  
Фактически это задача о покрытии отрезками: каждый бросок из i покрывает мячи в i−aᵢ .. i−1.  
Минимальное количество бросков — жадное покрытие слева направо.

**Сложность:** O(n)

Ввод
4
6 7 6 7
Вывод
4

Ввод
5
1 1 4 2 3
Вывод
8
