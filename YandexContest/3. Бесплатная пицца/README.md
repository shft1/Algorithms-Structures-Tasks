# 3. Бесплатная пицца

## Условие

Дино нашёл купон, который позволяет получить бесплатную пиццу. Радостный Дино побежал менять купон, который, как оказалось, не обменивают на пиццу. Купон — это массив чисел, который был получен простым способом шифрования. Пиццу можно получить, расшифровав этот массив.

Массив был зашифрован следующим образом:

1. Взяли два массива натуральных чисел от **1 до n**, упорядоченных по возрастанию, одинаковой длины **n**.
2. Каждый массив сдвинули **циклическим сдвигом вправо** один раз на некоторую величину **k** и **m** соответственно.
3. Числа **k** и **m** выбраны таким образом, что:  
   (1 ≤ k < 10⁶, 1 ≤ m < 10⁶) и (k ≤ m < n).
4. Далее все числа массивов сложили по одинаковым индексам так, что получили один массив:  
   `arr1[i] + arr2[i] = given_arr[i]`.

Чтобы получить бесплатную пиццу, нужно назвать всего лишь эти 2 числа **k** и **m**.

Помогите Дино получить бесплатную пиццу и найдите эти 2 числа.

---

## Формат ввода

- Первая строка — размер массивов **n** (2 ≤ n ≤ 10⁶).
- Вторая строка — зашифрованный массив чисел на купоне.

---

## Формат вывода

Выведите получившиеся числа **k** и **m**, разделённые пробелом.

---
