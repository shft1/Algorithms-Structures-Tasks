# 4. Спрятанные значения

## Условие

Вам достался файл с логами запросов и ответов. К сожалению, файл повреждён: структурированные данные в формате JSON оказались перемешаны с произвольным мусорным текстом. Необходимо восстановить как можно больше данных.

Формально, дана строка **S**. В ней могут встречаться корректные JSON-значения следующих типов:

- **Объект**: `{"key": "value", "n": 1}`
- **Массив**: `[1, 2, "abc"]`
- **Строка**: `"hello"`
- **Целое число**: `42`, `-7`, `0`

Гарантируется, что другие корректные JSON-типы (дробные числа, `true`, `false`, `null`) в данной задаче **не используются**.

Требуется выбрать такое множество **непересекающихся подстрок** **S**, что:

1. Каждая подстрока является корректным JSON-значением.
2. Никакие две выбранные подстроки не пересекаются (то есть не имеют общих символов).
3. Суммарная длина выбранных подстрок **максимальна**.

---

## Формат ввода

Одна строка **S** (длиной не более **50 000** символов).

---

## Формат вывода

- В первой строке — два числа через пробел:  
  **k** — количество выбранных JSON-значений  
  **max_length** — максимальная суммарная длина выбранных подстрок.
- В каждой из следующих **k** строк — два числа **lᵢ** и **rᵢ**:
  - **lᵢ** — начало подстроки (включительно)
  - **rᵢ** — конец подстроки (невключительно)
  - Отсчёт позиций с **0**
  - Порядок строк не важен.

---

## Примеры

### Пример 1

**Ввод:**
hello "a"bbbbbbbbb"c" world
Вывод
1 11
8 19

Ввод
data: {"key": "value"} and [1, 2, 3]
Вывод
2 25
27 36
6 22

Ввод
no json {} {"x": 1} ??? {"y": [2, 3]} end
Вывод
3 23
24 37
11 19
8 10
