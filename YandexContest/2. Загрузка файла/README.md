# 2. Загрузка файла

## Условие

Вы загружаете файл размером ровно **N** байт в объектное хранилище. Используемый протокол требует, чтобы размер каждого чанка был простым числом байт. Необходимо разбить файл на минимальное количество чанков.

Формально: представьте **N** в виде суммы минимального количества простых чисел. Слагаемые могут повторяться.

Если решений несколько, выведите любое.

---

## Формат ввода

Одно целое число **N** (2 ≤ N ≤ 1 000 000).

---

## Формат вывода

- В первой строке — одно число **k** — минимальное количество простых слагаемых.
- Во второй строке — **k** простых чисел через пробел, сумма которых равна **N**.

---
