# 5. Робот-курьер

# Задача: Робот-курьер в квадратном городе с метро

## Условие

Есть квадратный город, в котором работает робот-курьер. Город размечен как сетка из клеток. Каждая клетка (район) кодируется числом:

- **0** — пустая;
- **−1** — недоступна для робота;
- **натуральное число** — ценность посылки в этом районе;
- **−2, −3, …** — уникальный идентификатор станции метро (в этих клетках нет посылок).

Сеть метро задаётся списком соединений станций метро `(id₁, id₂)`. Станции объединяются в **ветки метро**.  
Ветка метро — это множество станций, между которыми существует путь по списку соединений.  
Робот может двигаться по этим веткам по определённым правилам (описаны ниже).

Робот реализует два маршрута:  
- **прямой** — из точки `(0, 0)` в точку `(n-1, n-1)`  
- **обратный** — из точки `(n-1, n-1)` в точку `(0, 0)`  

Посещая клетку района, он забирает в ней посылку. Посылка забирается из клетки **один раз** (даже если робот проходит через неё в обоих маршрутах).

Чтобы не создавать хаос, робот передвигается по следующим правилам:

### Прямой маршрут:
- может идти **вправо** `(i, j+1)` и **вниз** `(i+1, j)`;
- а также осуществлять **переход от станции метро id₁ к станции id₂**, если:
  - они на **одной ветке метро**;
  - и каждая из координат id₁ **не больше** соответствующей координаты id₂:  
    `(i₁ ≤ i₂, j₁ ≤ j₂)`.

### Обратный маршрут:
- может идти **влево** `(i, j-1)` и **вверх** `(i-1, j)`;
- а также осуществлять **переход от станции метро id₁ к станции id₂**, если:
  - они на **одной ветке метро**;
  - и каждая из координат id₁ **не меньше** соответствующей координаты id₂:  
    `(i₁ ≥ i₂, j₁ ≥ j₂)`.

Требуется вычислить **максимальную суммарную ценность**, которую робот может собрать, совершив две поездки: сначала по прямому маршруту, затем по обратному.  
Если **прямой** или **обратный** маршрут осуществить невозможно, следует вывести **0**.

---

## Формат ввода

- В первой строке **n** — размер стороны квадрата, описывающего город (1 ≤ n ≤ 500).
- В следующих **n** строках — по **n** чисел — коды каждого района / клетки (числа по модулю не превышают 10⁹).
- В следующей строке **m** — количество связей в метро (0 ≤ m ≤ 1000).
- В следующих **m** строках — связи: `id₁ id₂` (гарантируется, что связи корректны, указанные id существуют).

---

## Формат вывода

В единственной строке выведите одно целое число — **максимальную суммарную ценность** или **0**, если прямой или обратный маршрут невозможен.
